# 🍽️ Louis Buffet - Landing Page

Esta é uma landing page desenvolvida para o **Louis Buffet**, uma empresa de eventos com mais de duas décadas de tradição em Assis-SP. O projeto foi criado para modernizar a presença digital do buffet e facilitar o contato de novos clientes.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização personalizada, animações e responsividade.

## 📱 Funcionalidades

- **Design Responsivo**: Adaptado para visualização em desktops, tablets e smartphones (Mobile First).
- **Integração com WhatsApp**: Botões de chamada para ação (CTA) que direcionam diretamente para o orçamento.
- **Navegação Suave**: Implementação de *Smooth Scroll* para melhor experiência do usuário.
- **Animações de Entrada**: Efeito visual de carregamento para destacar a proposta de valor.
